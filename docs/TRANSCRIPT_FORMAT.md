# Tasklet Transcript App — Transcript Format

## Standard document structure

Cleaned meeting transcripts normally contain:

1. Document title
2. Meeting title
3. Date
4. Start time
5. Duration
6. Participants
7. Subject
8. Editorial note
9. Meeting summary
10. Key decisions and working assumptions
11. Follow-up items
12. Open questions
13. Practical implementation checklist
14. Commercial and partner notes
15. Cleaned transcript
16. Topic headings
17. Timestamped dialogue
